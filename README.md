# PolomonGo
Simple CLI to get candlestick data from Poloniex for ETH/BTC, put it in a MongoDB and print the collection entries.

List of pairs :
    btc_etc,
    btc_eth,
    btc_ltc,
    btc_rep,
    btc_xmr,
    btc_xrp,

List of chart patterns:
    SMA (20 day),
    BB (20 day)
